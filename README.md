1.Project Overview

The HR Analytics Dashboard is an interactive Power BI report designed to analyze employee attrition patterns and workforce trends. The dashboard helps HR teams understand factors influencing employee turnover and supports data-driven decision-making for improving employee retention and organizational performance.

2. Objective

The goal of this project is to:

Monitor employee attrition and workforce metrics.
Identify trends based on age, salary, education, and job roles.
Support HR professionals in making informed decisions to improve retention.
3. Tech Stack

The dashboard was built using the following tools and technologies:

📊 Power BI Desktop – Data visualization and report creation.
📂 Power Query – Data cleaning and transformation.
🧠 DAX (Data Analysis Expressions) – Creation of calculated measures and KPIs.
📁 Excel Dataset – Source data for employee information.
🖼️ PNG – Dashboard preview image.

5. Key Features
Business Problem

Employee attrition affects productivity and increases hiring costs. HR teams need an efficient way to identify factors contributing to employee turnover.

Goal of the Dashboard
Track workforce KPIs.
Analyze attrition across different employee segments.
Provide insights to support retention strategies.

7. Dashboard Highlights
Attrition by Education

Analyzes employee attrition across different educational backgrounds.

Attrition by Age Group

Identifies age groups with higher attrition rates.

Attrition by Salary Slab

Shows the relationship between salary ranges and employee turnover.

Attrition by Years at Company

Analyzes employee retention based on tenure.

Attrition by Job Role

Highlights job roles experiencing higher attrition.

Department Filter

Allows users to analyze HR metrics across different departments.

8. Business Insights
Employees with lower salary ranges experienced higher attrition.
The 26–35 age group showed the highest employee turnover.
Sales-related roles contributed significantly to attrition.
Workforce trends can help HR teams develop targeted retention strategies.
